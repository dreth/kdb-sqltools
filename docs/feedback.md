# Feedback

Use the Command Palette:

- `kdb+: Report Bug`
- `kdb+: Request Feature`
- `kdb+: Give Feedback`

GitHub issue links:

- [Bug report](https://github.com/dreth/kdb-sqltools/issues/new?title=Bug%3A%20&labels=bug)
- [Feature request](https://github.com/dreth/kdb-sqltools/issues/new?title=Feature%20request%3A%20&labels=enhancement)
- [General feedback](https://github.com/dreth/kdb-sqltools/issues/new?title=Feedback%3A%20&labels=feedback)

For bug reports, include:

- VS Code version.
- SQLTools version.
- `kdb-sqltools` version.
- OS.
- kdb+/q version.
- Connection target type: local, remote, or SSH tunnel.
- The q text that reproduces the issue, reduced if possible.
